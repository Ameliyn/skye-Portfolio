This is a collection of my work in Computer Architecture and Assembly. It is designed to be viewed in a linux environment.


It contains:

1. bignum
    This is a project to represent big numbers using bits.

2. bitlib
    This is a project that contains coding the bitwise operators (AND, OR, NOR, etc)

3. LinkedList
    This is a project to create a linkedlist being conscious of memory.

4. MatrixMultiplication
    This project entails doing matrix manipulation.

5. tinycalc
    This is a project to make a tiny calculator that reads in ASCII.
