This is a compilation of the work done in the Basic Graphics class.



It Contains:

1. CompletedLabs
    This is a collection of iterated labs that grow and build on the previous labs.

2. SpinningTorus
    This is a project that shows a 3D spinning torus that has a light model and is rendered using painters algorithm.

3. RocketLaunch
    This is a project that shows a 2D rocket that launches. It was mostly a proof of concept for 2D matrix manipulation.

4. UnsortedMess
    This is an unsorted mess of partially completed labs and random other files used over the year.
